#### 연소의 3요소
- 연료
- 공기
- 점화원
#### 연소의 분류(종류)
- 표면연소 : 목탄, 코크스 표면이 산소와 반응
- 분해연소 : 고체연료 연소방식 (석탄, 목재)
- 증발연소 : 액체연료 연소방식 
- 확산연소 : 기체연료 연소방식
- 자기연소 : 산소 공급없이 연소 (제5류 위험물)
#### 연소점
- 인화점 : 외부 점화원에 의해 가연성 가스가 연소 시작하는 최소 온도
- 발화점 : 스스로 연소 시작하는 최소 원도
- 발화지연 : 가열하기 시작하여 발화까지 걸린 시간
#### 가스발화의 외부점화원
- 화염
- 정전기
- 마찰열
- 단열압축
- 전기불꽃
- 충격불꽃
#### 발화의 요인
- 온도
- 압력
- 조성
- 용기의 크기
#### 자연발화의 원인
- 산화열에 의한 발열
- 분해열에 의한 발열
- 중합열에 의한 발열
- 미생물에 의한 발열
- 흡착열에 의한 발열
#### 발화점이 낮아질 수 있는 조건
- 압력이 높을수록
- 산소농도가 높을수록
- 발열량이 클수록
- 반응활성도가 클수록
- 열전도율이 낮을수록
- 분자량이 클수록
#### 기체의 반응속도 관련 인자
- 압력
- 온도
- 농도
- 촉매
#### 폭발의 종류
- 물리적 폭발
- 화학적 폭발
#### 물리적 폭발의 종류
- 증기폭발 : 보일러 폭발
- 금속선폭발 : 알루미늄 전선에 과전류
- 고체상 전이 폭발 : 고상 전이
- 압력폭발 : 고압가스 용기 폭발
#### 화학적 폭발
- 산화폭발 : 산화반응 폭발
- 분해폭발 : 고압에서 아세틸렌, 산화에틸렌, 히드라진, 오존 분해폭발
- 중합폭발 : 시안화수소, 산화에틸렌, 부타디엔, 염화비닐 중합반응
- 촉매폭발 : 염소에 직사광선 촉매로 염소폭명기
- 분진폭발 : 금속분, 소맥분, 전분
#### 블레이브(BLEVE), 증기운 폭발
- BLEVE : 탱크 외부 화재로 약한 부분이 터지면서 유출가스가 파어어볼을 형성
- 증기운 폭발 : 대량의 가연성 가스가 대기 중에 공기와 혼합된 상태에서 점화원에 의해 급격히 폭발
#### 증기운폭발에 영향을 주는 인자
- 방출된 물질의 량
- 증기운이 퍼진 거리
- 점화원의 위치
- 점화 확률
- 폭발효율
#### 폭굉
- 연소속도가 음속보다 빠른 경우 (1000~3500m/s)
#### 폭굉 유도거리와 짧아지는 조건
- 연소가 폭굉까지 이어지는데 걸리는 거리 (짧을수록 위험)
- 가스의 연소속도가 클수록
- 관 저항 있거나 관지름이 작을수록
- 압력이 높을수록
- 점화원의 에너지가 클수록
#### 방폭구조 종류
- 내압방폭구조(d) : 용기가 폭발에 견디도록
- 압력방폭구조(p) : 용기 내부에 보호가스 충전
- 유입방폭구조(o) : 용기 내부에 절연유 주입
- 안전증방폭구조(e) : 기계/전기적으로 안전도를 증가
- 본질안전방폭구조(ia, ib) : 시험을 통해 안전이 검증된 구조
- 특수방폭구조(s) : 그외 구조로 시험을 통해 검증된 구조
#### 가스/내압방폭구조의 폭발등급 분류 (가스/전자기기)
- A(IIA) : 최대 안전틈새 0.9 mm 이상
- B(IIB) : 최대 안전틈새 0.5mm 초과 ~ 0.9mm 미만
- C(IIC) : 최대 안전틈새 0.5mm 이하
#### 가스/본질안전방폭구조의 폭발등급 분류 (가스/전자기기)
- A(IIA) : 최소 점화전류비 0.8 초과
- B(IIB) : 최소 점화전류비 0.45mm 이상 ~ 0.8mm 이하
- C(IIC) : 최소 점화전류비 0.45mm 미만
#### 가스의 발화온도에 따른 온도등급
- 450도 초과 : T1
- 300 ~ 450 : T2
- 200 ~ 300 : T3
- 135 ~ 200 : T4
- 100 ~ 135 : T5
- 85 ~ 100 : T6
#### 가연성가스의 폭발 우려가 있는 장소
- 1종장소 : 종종 가연성가스가 체류하여 폭발우려가 있는장소
- 2종장소 : 사고로 인해 가연성가스가 누출된 위험이 있는장소
- 0종장소 : 상시 가연성가스가 체류(폭발 하한계 이상)하는 장소
#### 불활성화 작업
- 불활성화 가스를 주입 산소 농도를 최소산소농도 이하로 떨어뜨리는 작업
- 진공퍼지 : 용기를 진공시키고 불활성화 가스 주입
- 압력퍼지 : 불활성화 가스를 주입 가압, 방출 반복
- 스위프 퍼지 : 한쪽은 불활성가스 주입 반대쪽은 방출 (저장탱크)
- 사이펀퍼지 : 용기에 물을 채우고 뺴면서 불활성가스 주입
#### 정성적 위험성 평가기법
- 체크리스트법 : 위험 상황을 **목록화**하여 누락 없이 점검
- 사고예상질문분석(WHAT-IF법) : "만약 ~한다면?"이라는 질문을 통해 공정의 위험과 결과를 예측
- 위험과 운전분석(HAZOP) : 정상이 아닌 모든 경우의 수를 따져보는 것
#### 정량적 위험성 평가기법
- 작업자 실수 분석 : 인간의 실수를 고장에 미치는 영향으로 숫자로 정확히 계산
- 결함수분석(FTA : Fault Tree Analysis) : 결과로부터 원인을 찾는 Top Down분석
- 사건수분석(ETA : Event Tree Analysis) : 원인으로부터 결과를 찾는 Bottom-up분석
- 원인결과분석(CCA) : FTA+ETA 결합한 통합기법
