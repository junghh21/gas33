#### 가연성 가스 가스 검출기법
- 안전등형 : 가스등의 불꽃 모양 변화를 관찰
- 간섭계형(광파굴절) : 가스의 빛 **굴절률**을 이용, 광학식 검출기
- 열선형(접촉연소식) : 가스의 열선 접촉 **산화(연소)**로 인해 **전기 저항이 증가**
- 반도체식 : 가스가 반도체 표면에 흡착하면 저항이 변화
#### 가스분석기 종류
- 화학적 가스 분석기 : 연소열, 용액 흡수, 고체 흡수
- 물리적 가스 분석기 : 열전도율, 빛의 간섭, 저항, 적외선 흡수
#### 가스의 흡수 분석법
- 오르자트법
- 헴펠법
- 게겔법
#### 가스의 흡수 분석 흡수 순서, 흡수제
- $CO_2 \rightarrow C_nH_m \rightarrow O_2 \rightarrow CO$
- $CO_2$ : KOH(수산화칼륨) 수용액
- $C_nH_m$ : 발연황산($H_2SO_4$)
- $O_2$ : **알칼리성 피로가롤** 용액
- $CO$ : **암모니아성 염화제1구리** 용액
#### 가스 크로마토그래피
- 운반가스가 컬럼을 통과하며 고정상에 의한 지연을 측정
- 다양한 가스의 검출이 가능
- 감도가 좋다 (미량가능, 분리능력 우수)
- 응답속도가 느리다. (수분~수십분)
- 동일가스 연속측정이 어렵다. (컬럼 퍼징 필요)
#### 가스 크로마토그래피 캐리어가스
- 수소 ($H_2$)
- 아르곤 ($Ar$)
- 헬륨 ($He$)
- 질소 ($N_2$)
#### 가스크로마토그래피 흡착제(충전제)
- 활성탄
- 활성알루미나
- 실리카겔
- 제올라이트
#### 가스 크로마토그래피 검출기
- 열전도형 검출기 (TCD) : 유기,무기 일반적
- 수소 이온화 검출기 (FID) : 탄화수소
- 전자포획 이온화 (ECD) : 할로겐, 산소
- 불꽃 광도형 (FPD) : 인, 황
#### 접촉식 온도계 종류
- 유리제 봉입식 온도계
- 알콜유리 온도계
- 바이메탈 온도계 : 이종금속 팽창차이 이용
- 압력식 온도계 : 유체의 체적변화
- 전기식 저항 온도계 : 백금, 니켈 저항
- 전기식 서미스터 온도계 : 저항값이 변하는 반도체
- 열전대 온도계 : 제백효과 (기전력)
#### 열전대 온도계 종류
- 백금-로듐 (P-R)
- 크로멜-알루멜 (C-A)
- 철-콘스탄트 (I-C)
- 동-콘스탄트 (C-C)
#### 비접촉식 온도계 종류
- 광고온도계: 표준 전구의 필라멘트 밝기와 비교 (고온 용광로)
- 광전관식 온도계 : 광전소자로 온도 측정
- 방사온도계 : 스테판볼츠만 법칙 (복사에너지의 총량)
- 색온도계 : 2개의 파장을 비교
#### 압력계 1차 압력계
- 액주식 압력계
- 침종식 압력계 : 부력 이용 (단종식, 복종식)
- 자유 피스톤식(분동식) 압력계 : 부르동관 압력계 교정용
#### 액주식 압력계 종류
- 단관식
- U자관식
- 경사관식
#### 압력계 2차 압력계
- 탄성 압력계
- 전기식 압력계
#### 탄성 압력계 종류
- 부르동관 압력계
- 벨로즈식 압력계
- 다이어프램 압력계
- 캡슐식 압력계
#### 부르동관의 재질
- 저압 : 황동, 청동
- 고압 : 스테인레스강, 니켈강
#### 전기식 압력계(급격한 변화, 원격) 종류
- 전기저항 압력계
- 피에조 압력계
- 스트레인 게이지
#### 유량계의 종류
- 직접법 : 유체의 부피나 질량을 직접 측정
- 간접법 : 유속을 측정 (베르누이 정리)
#### 유량계 직접식 유량계 특징
- 정도가 높다
- 고점도 유체 적합
- 맥동의 영향을 적게 받는다.
- 여과기 설치 (이물질 유입 차단)
#### 직접식 유량계 종류
- 오벌식
- 루츠식
- 로터리피스톤식
- 로터리 베인식
- 습식 가스미터
- 왕복 피스톤식
####  간접식 유량계 종류
- 차압식 유량계 : 베르누이 정리
- 유속식 유량계 : 임펠러식, 피토관식, 열선식
- 초음파 유량계 : 도플러 효과
- 전자식 유량계 : 페러데이의 전자유도법칙
- 면적식 유량계 : 부자식(플로트식), 로터미터
- 와류식 유량계 : 와류의 주파수 특성
#### 유량계 간접식 차압식 유량계 종류
- 오리피스 미터 : 구멍이 있는 원반을 배관에 설치
- 플로어노즐 : 오리피스와 벤투리의 중간 형태
- 벤투리미터 : 압력 손실이 가장 적음
#### 직접식 액면계 종류
- 유리관식 액면계
- 부자식(플로트식) 액면계
- 검척식 액면계 : **측정척(자)이나 줄자**를 이용해 직접 재는 방식
#### 간접식 액면계 종류
- 정전용량식
- 초음파식
- 압력식
- 편위식
- 기포식
- 다이어프램식
