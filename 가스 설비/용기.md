## 용기 일반
#### 오토클레이브
- 정의 : 고압과 고온에 견딜 수 있는 용기
#### 오토클레이브 종류
- 교반형
- 진탕형
- 회전형
- 가스 교반형
#### 진탕형 오토클레이브 특징
- 회전축이 없어 밀페가능
- 구조가 단순
- 용기를 흔들어야 하므로 에너지 많이 들고 기계적 부담
- 고점도 혼합은 불리 (교반형)
## 압력용기 (저장탱크)
#### 원통형 저장탱크 구조
- 수평형, 수직형
- 구형탱크에 비해 두께가 두껍다.
- 수평형이 안정, 수직형은 두껍게 (바람, 지진)
- 경판의 종류 : 접시형, 타원형, 반구형
- 부속기기 : 유체 입출구, 드레인밸브, 안전밸브, 액면계, 온도계, 압력계
#### 구형 저장탱크 구조
- 표면적 작고 단열성이 좋다.
- 강도가 높다.
- 기초가 간단, 설치비 부담 적고
- 외관이 안정적 
- 부속기기 : 유체 입출구, 상하 맨홀, 안전밸브, 온도계, 압력계
#### LNG 저장시설의 방호 종류
- 단일방호 : 저장탱크 단열 + 방류둑
- 2중방호 : 2중 저장 탱크 
- 완전방호 : 2중 저장 탱크 + BOG 벤트 처리
## 충전용기
#### 충전용기 재료의 구비 조건
- 내식성, 내마모성
- 가볍고, 강도가 높을 것
- 연성, 전성
- 가공성 좋을 것
#### 충전용기의 종류
- 이음매 없는 용기
- 용접 용기
#### 충전용기 이음매 없는 용기 특징
- 비싸다
- 고압에 적합
- 내압이 균일하게 분포
- 두께가 균일하지 않을 수 있다.
- 함량 : 탄소 0.55%, 인: 0.04%, 황: 0.05%
#### 충전용기 용접 용기 특징
- 저렴하다
- 두께가 균일하다.
- 용기 형태, 치수 자유롭다.
- 함량 : 탄소 0.33%, 인: 0.04%, 황: 0.05%
#### 충전용기 이음매 없는 용기 제조법
- 만네스식, 에르하트식, 디프드로잉식
#### 충전용기 초저온용기
- -50이하 액화가스
- 단열재나 냉각으로 상용온도 유지 조치
- 알루미늄, 오스테나이트 스테인레스강
#### 저온장치용 금속재료
- 응력이 적은 부분 : 동, 알루미늄
- 응력이 있고 약간 낮은 온도 : 탄소강
- 응력이 있고 극저온 : 오스테나이트계 스테인레스강
#### 이동식 초저온 용기 취급시 주의 사항
- 충격을 피한다.
- 직사광선, 눈, 비 피한다.
- 기름묻은 장갑 ,면장갑 사용 피한다.
- 통풍 불량 지하실 보관 피한다.
- 충전용기 잔가스 용기는 분리
#### 초저온 액화가스 취급중 발생할 수 있는 사고
- 급격한 기화로 인한 압력 변화
- 저온에 의한 물리적 변화
- 동상
- 질식
#### 용기 밸브 충전구 형식에 따른 분류
- A형 : 수나사
- B형 : 암나사
- C형 : 나사가 없는 방식
#### 용기 밸브 충전구 나사형식에 따른 분류
- 왼나사 : 가연성가스 
- 오른나사 : 가연성가스 외 (암모니아, 브롬화메탄)
#### 충전용기의 안전밸브 방식
- LPG 용기 : 스프링식
- 염소, 아세틸렌 가스 용기 : 가용전식
- 산소, 수소, 질소, 액화이산화탄소 용기 : 파열판식
- 초저온 용기 : 스프링식, 파열판식 2중
#### 스프링식 안전밸브 특징
-  빠른 응답 속도
- 구조가 단순
- 고압적용가능
- 밀폐식 구조가 가능하여 외부 증기 누출을 방지
#### 가용전식 안전밸브 사용 가스
- 염소,아세틸렌,산화에릴렌용기
#### 파열판식 안전밸브 특징
-  산소,수소,질소,아르곤
- 압력변화가 많은 곳에 적당
- 1회 사용 후 교체 필요
- 구조가 간단
- 부식성 유체에도 사용가능
- 밸브시트 누설이 없음
- 스프링식과 파열판식 2중 안전밸브 - 초저온용기
## 용기 검사
#### 푸아송 비
- 재료가 인장(당김)을 받을때 **가로 세로 변형**의 비율
- 금속재료에서 약 0.3으로 **탄성한계** 이내 (늘려도 다시 돌아옴)
- 푸아송 수 : 푸아송 비의 역수
#### 항구증가율 계산
- 공식 : $항구증가율 = \frac{항구증가량}{전증가량}$
#### 항구증가율 합격기준
- 신규검사 : 항구증가율 10% 이내
- 재검사 : 항구증가율 10% 이내(질량검사 95% 이상), 6%이내(질량검사 : 90~95%)
#### 충전용기의 시험압력
- 압축가스 : 35도 최고충전압력(FP), 기밀 : FP 1배, 내압(TP) : FP 1.6배, 안전밸브 : TP 0.8배
- 아세틸렌 가스 : 15도 최고충전압력(FP), 기밀 : FP 1.8배, 내압(TP) : FP 3배, 안전밸브 : 가용전식 105+-5도
- 초저온 용기 : 최고상용압력, 기밀 : FP 1.1배, 내압 : FP 1.6배, 안전밸브 : TP 0.8배
- 액화가스용기 : 최고충전압력, 기밀 : FP 1배, 액화가스별로 규정, 안전밸브 : TP 0.8배
#### 신규검사에 합격한 용기 각인
- 제조업자 이름
- 가스의 명칭
- 용기 번호
- V : 내용적
- W : 용기의 질량 (밸브 제외)
- TW : 아세틸렌 용기 질량 (다공물질, 용제, 밸브포함)
- 내압시험 합격연월
- TP : 내압시험압력 (Mpa)
- FP : 최고충전압력 (Mpa)
#### 초저온용기 신규 검사 추가항목
- 단열성능시험
- 용접부충격시험
#### 충전용기 재검사를 받아야하는 경우
- 재검사 기간도래
- 손상
- 열영향
- 합격표시 훼손
- 충전가스 변경
#### 이음매없는 용기의 검사
- 외관검사
- 내압시험
- 기밀시험
- 파열시험
- 압궤시험 : 용기를 강재 쐐기로 눌러 변형과 파손을 확인
#### 이음매없는 용기의 재검사 항목
- 외관검사
- 음향검사
- 내압검사
#### 수조식 내압기의 특징
- 소형용기에 적합
- 팽창이 정확히 측정
- 결과의 신뢰성이 높다.
#### 용기별 부속품 기호
- AG : 아세틸렌 용기 부속
- PG : 압축가스 용기 부속
- LG : 액화가스 용기 부속 (LPG 제외)
- LPG : LPG 용기 부속
- LT : 초저온 용기 부속
#### 용기의 도색, 글자색 (공업용)
- 산소 : 녹색, 백색 (의료용 백색, 녹색 글자)
- 수소 : 주황색, 백색
- 이산화탄소 : 청색, 백색
- LPG : 밝은 회색, 적색
- 아세틸렌 : 황색, 흑색
- 암모니아 : 백색, 흑색
- 염소 : 갈색, 백색
- 이외 : 회색, 백색
- 가연성가스 '연', 독성가스 '독'
- 선박용 : LPG 띠 2줄 '선박용'
## 비파괴 검사
#### 도시가스 배관 용접시 검사 방법
- 육안검사와 방사선투과검사
- 안되면 초음파검사와 자분탐상검사
#### 금속 용기 비파괴 검사 종류
- 육안검사
- 음향검사 : 두드려봐서
- 침투탐상검사 : 침투액을 닦고 현상제로 결함 모양 확인 (염료법, 형광법)
- 자분탐상검사 : 자화시키고 자분의 모양을 확인
- 방사선투과검사 : X선 투과 (용접부 검사)
- 초음파탐상검사 : 초음파 반사파로 내부결함 모양 확인
- 와류검사 : 전류 흐름 이용
- 전위차법 : 전위차로 결함 깊이 확인
#### 비파괴 검사 방사선투과검사 특징
- 내부 결함 검출 가능
- 기록 유지 가능
- 장비가 비싸다.
- 방호에 유의
#### 자분탐상시험(Magnetic Particle Testing, MPT) 장점
- 작업이 신속하고 간단
- 결함 눈으로 확인 가능
- 비교적 저렴한 비용 – 경제적.
- 다양한 자성체 표면에 적용 가능
#### 자분탐상시험(Magnetic Particle Testing, MPT) 단점
- 비자성체(알루미늄, 구리 등)에는 사용할 수 없음.
- 내부 결함 검사 불가능
- 검사 후 탈자 과정 필요, 후처리 필요
- 전원 필요
#### 초음파 탐상검사 원리에 의한 분류
- 공진법
- 투과법
- 펄스반사법
#### 초음파 탐상검사장점
- 감도가 높으므로 미세한 결함의 검출 가능
- 검사시 시험편의 한쪽면 만 사용 
- 내부 불연속 위치, 크기, 방향, 모양을 정확하게 측정 가능
- 방사선 투과 검사법과 같이 인체 및 환경오염이 없다
- 탐상 결과를 바로 얻을 수 있으며 자동화가 가능
- 투과력이 높으므로 두꺼운 재질이나 깊은 곳의 결함도 찾기 용이
#### 초음파 탐상검사단점
 - 시험체 표면이 깔끔, 재질이 균일, 두께가 두꺼워야함
 - 불감대가 있다
 - 초음파 방향에 따라 감지 영향이 크다
 - 검사자의 속련도가 필요
 - 표면결함 검출은 떨어진다.
