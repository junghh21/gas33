## 연소기구

#### 공기와 가스의 혼합방식에 따른 연소방식
- 적화식
- 분젠식
- 세미분젠식
- 전1차공기식
#### 적화식
- 예혼합없이 2차 공기로 연소
- 확산연소, 연소 속도가 느림
- 고열을 얻기 힘들다.
- 역화 우려 없다
- 불완전 연소 우려 (연소실이 작으면)
#### 분젠식
- 노즐을 통해 분출된 가스가 1차 공기를 공급 받고 이후 염공을 통해 연소하면서 2차 공기를 공급 받는 방식
- 연소속도가 빠르고 화염이 낮다
- 연소 온도가 높다.
- 연소실이 작아도 된다.
- 리프팅 발생
- 역화 발생 (공기 조절기 조정필요)
#### 세미분젠식
- 적화식과 분젠식의 혼합
- 역화우려가 적다.
- 고온을 요하는 곳에 적당하지 않다.
#### 전1차 공기식
- 송풍기로 공기를 압입
- 버너를 어떤 방향으로도 설치 가능
- 구조가 복잡하고 비싸다.
- 압력조정기 필요
#### 연소기구의 조건
- 완전연소
- 열효율이 좋을 것
- 취급이 쉽고 안전할 것
#### 염공이 갖추어야할 조건
- 모든 염공에 빠르게 확산
- 불꽃이 안정적
- 배열이 적당
- 청소가 용이
#### 연소기구 이상현상
- 역화 : 분출압력이 낮고 연소속도가 빠를 때 불꽃이 공급관을 타고 들어가는 현상
- 선화 : 분출압력이 높고 연소속도가 느릴때 불꽃이 염공보다 높은 곳에 형성되는 현상
- 블로우오프 : 불꽃이 염공에서 떨어져 꺼져버리는 현상
- 옐로우팁 : 공기부족 불완전연소로 온도가 낮아 불꽃이 노랗게 보이는 현상
#### 연소기구 공기부족(불완전연소) 원인 
- 공기 공급 부족
- 배기 불량
- 환기 불량
- 가스 조성 불량
- 연소기구 부적합
- 기구 냉각
#### 역화의 원인
- 염공이 크게 되었다.
- 공급가스의 압력이 저하
- 버너의 과열
#### 선화의 원인
- 염공이 작다
- 공급압력이 높다
- 2차 공기 부족 (배기, 환기)
- 1차 공기 과다
#### 연소안전장치 ?
- 불이 꺼졌을때 공급을 자동 차단
#### 연돌효과
- 굴뚝이 높을수록 밀도차가 커서 연기가 잘 빠져나감
#### 반밀폐형 가스보일러의 역풍방지구의 역할
- 배기가스 역풍방지
- 연돌효과로 과도한 공기 흡입방지
- 연소기구 열효율 저하 방지
- 기구실내 공기 흡입해서 환기
#### 반드시 전용 보일러실에 설치해야 하는 가스보일러
- 반밀폐식
- 강제배기식
#### 개방형 연소기실에 설치해야 하는 것
- 환기구
#### 반밀폐형 연소기를 설치하는 곳에 설치해야 하는 것
- 급기구, 배기통
#### 배기통 재료로 사용할 수 있는 재료
- 스테인레스강
- 내열, 내식 재료
#### 보일러를 전용보일러실에 설치하지 않아도 되는 경우
- 밀폐식 (강제급배기식)
- 옥외설치
- 전용급기통의 강제배기식
#### 연소가스의 급 배기 방식에 따른 분류
- 개방형 : 실내 흡입, 실내 배기
- 반밀폐형 : 실내 흡입, 외부 배기
- 밀폐형 : 외부 흡입, 외부 배기
#### 밀폐식 보일러 설치 장소의 환기구 크기
- $1M^2$당 $300cm^2$ 
#### 강제배기식 가스보일러의 배기통 터미널 설치 기준
- 내식성 구조물 1.6cm 이하 (새, 쥐 방지)
- 60cm 이내 가연성 구조물 금지
- 60cm 이내 실내 개구부 금지
## 부취제  

#### 부취제 종류
- T.H.T(썩은양파)
- T.B.M(석탄가스냄새)
- D.M.S(마늘냄새)
#### 부취제의 주입방식
- 액체주입식
- 증발식
#### 부취제 주입방식 - 액체주입식
 - 펌프 주입방식 : 펌프 이용 직접 주입(농도 정밀 제어 가능)
 - 적하 주입방식 : 가스가 부취제 탱크 통과
 - 미터연결 바이패스 방식 : 유량계 바이패스를 통해 부취제 탱크 통과
#### 부취제 주입방식 - 증발식
 - 바이패스식
 - 위크식 : 부취체 액상에 심지를 통해 증발을 가속하는 방식
#### 부취제의 구비조건
 - 기능 (냄새 독특, 낮은농도1/1000에서 검출가능)
 - 인체 (독성, 물에 녹지않음)
 - 설비 (가스관 부식, 흡착)
 - 제거 (완전연소, 토양투과, 화학적으로 안정)
#### 부취제의 냄새측정 방법
- 오더미터법 : 장비이용 계측
- 주사기법 : 주사기 이용 비율 측정
- 냄새주머니법 : 주머니 이용 비율 측정
- 무취실법
#### 부취제의 제거방법
- 활성탄 흡착
- 화학적 산화 (차아염소산나트륨)
- 연소법

## 가스미터
#### 막식 가스미터
- 가격이 저렴
- 유지관리 쉬움
- 대용량은 설치면적 크다
#### 습식 가스미터
- 계량이 정확
- 수위조절등 관리 필요
- 설치면적이 크다.
- 실험실용, 기준용
#### 루츠형 가스미터
- 대유량 측정에 적합
- 중압가스 개량 가능
- 설치면적이 적다
- 여과기 필요
- 적은 유량은 부동의 우려
#### 가스미터 설치장소 기준
- 화기와 2m이상 우회
- 직사광선 대비 격납상자
- 바닥에서 1.6~2m
- 전기계량기와 60cm 이상

## 도시가스의 연소성
#### 도시가스 연소속도
$$C_p = K\frac{1.0H_2+0.6(CO+C_mH_n)+0.3CH_4}{\sqrt{d}}$$

#### 웨버지수
$$WI = \frac{H_g}{\sqrt{S}}$$
- **$H_g$**: 가스의 총발열량 ($kcal/Nm^3$)
- **$S$**: 가스의 비중 (공기 = 1)

| **구분**          | **도시가스 (LNG / 메탄 주성분)**             | **LPG (프로판 주성분)**                   |
| --------------- | ----------------------------------- | ----------------------------------- |
| **발열량 ($H_g$)** | 약 $10,500 \sim 11,000 \, kcal/Nm^3$ | 약 $24,000 \sim 26,000 \, kcal/Nm^3$ |
| **비중 ($S$)**    | 약 $0.55 \sim 0.65$ (공기보다 가벼움)       | 약 $1.5 \sim 2.0$ (공기보다 무거움)         |
| **웨버지수 ($WI$)** | **약 $13,000 \sim 14,000$**          | **약 $18,000 \sim 20,000$**          |
