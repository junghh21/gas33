#### 카르노 사이클 순환과정
- **등-단-등-단** (등온팽창 → 단열팽창 → 등온압축 → 단열압축)
- **등온 팽창** ($T_H$ 일정) : 고온 열원으로부터 **열($Q_H$)을 흡수** (부피 증가, 압력 감소)
- **단열 팽창** ($Q=0$) : 외부로 **일($W$)을 함**, 온도는 강하 ($T_H \to T_L$) (부피 증가, 압력 급감)
- **등온 압축** ($T_L$ 일정) : 저온 열원으로 **열($Q_L$)을 방출** (부피 감소, 압력 증가)
- **단열 압축** ($Q=0$) : 외부로부터 **일($W$)을 받음**, 온도 상승 ($T_L \to T_H$) (부피 감소, 압력 급증)
#### 스털링 사이클 순환과정
- **등온 팽창** : 고온 열원으로부터 열을 흡수하며 일을 함 ($T$ 일정, $V$ 증가, $P$ 감소)
- **등적 냉각** : **재생기(Regenerator)**에 열을 저장하며 온도 하강** ($V$ 일정, $T$ 하강, $P$ 감소)
- **등온 압축** : 저온 열원으로 열을 방출하며 압축됨 ($T$ 일정, $V$ 감소, $P$ 증가)
- **등적 가열** : 재생기에 저장했던 열을 다시 흡수하며 온도 상승 ($V$ 일정, $T$ 상승, $P$ 증가)
#### 줄 톰슨 효과
- **고압**의 기체가 **교축밸브**를 통해 급격히 **단열팽창**하면 **온도**가 내려가는 현상
#### 5대 액화 사이클 비교
- **린데(Linde)** : **단순 줄-톰슨**, 효율 최저
- **클로드(Claude)** : 팽창기
- **카피자(Kapitza)** : 터빈 (현대식)
- **헤일란트(Heylandt)** : 클로드식의 고압 버전
- **캐스케이드(Cascade)** : **다단 냉동**, 효율 최고, 복잡
- **필립스(Philips)** : **스털링 엔진**, 재생기, 소형

## 공기(가스)액화 분리 장치
#### 공기(가스)액화 분리 장치의 구성요소
- 한랭발생장치
- 정류장치
- 불순물제거장치
#### 공기(가스)액화 분리 장치의 주요 폭발 원인
1. 공기 흡입구를 통해 아세틸렌($C_2H_2$)의 혼입 (가장 위험)
2. 압축기 윤활유가 분해 탄화수소류($CH_4, C_2H_6$ 등)의 농축
3. 공기 중에 질소산화물($NO_x$) 혼입
4. 액체 산소중에 오존($O_3$) 혼입  
#### 공기(가스)액화 분리 장치의 폭발 방지 대책
1. 아세틸렌 발생  없는 곳에 공기 흡입구를 설치
2. 양질의 윤활류 사용, 사염화탄소(기름때 제거제) 청소 
3. 고성능 필터 사용 (여과기)
4. 액체공기 퍼징
#### 공기(가스)액화 분리 장치의 수분제거 이유
저온에서 공기중 수분은 얼음, 이산화탄소는 드라이아이스가 되어 배관이나 밸브 파손
#### 공기(가스)액화 분리 장치의 불순물 제거 방법
1. 먼지, 이물질 : 고성능 여과기
2. 이산화탄소 : 수산화나트륨 수용액으로 흡수  -  $2NaOH + CO_2 \rightarrow Na_2CO_3 + H_2O$
3. 수분 : 실리카겔 건조기
4. 아세틸렌 : 아세틸렌 흡착기
#### 공기의 비점과 상태변화
- **비점 높음 => 기화 어려움, 액화 쉬움** (액화 속도 : 산소 > 질소)
- **비점 낮음 => 기화 쉬움, 액화 어려움** (기화 속도 : 산소 < 질소)
- 정류탑에서 질소(상부), 산소(하부)

- 상압단열법 : 단열재 충전

- 같은 두께에서 고 진공단열법보다 효과적
- 단열재, 기체는 저온에서 열전도율이 낮아 유리-
- 단열층으로 인해 지지력 생성
- 높은 진공도에서 효과적

- 샌다셀
