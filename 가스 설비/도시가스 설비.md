#### 도시가스의 원료
- 천연가스 : 천연가스, 석유시추 통해 직접 얻는 가스
- 액화천연가스 : 저온에서 액화된 가스
- 정유가스 : 석유 정제과정 부산물
- 나프타 : 열분해, 수소화분해
- LPG : 생가스, 공기혼합, 변성가스, 도시가스혼합
- CO, CO2 : 수소 접촉분해
# 도시가스의 제조

#### 천연가스의 수송, 액화(LNG) 전처리
- 제진
- 탈유
- 탈황
- 탈탄산
- 탈수
#### 도시가스의 가스화 방식
- 열분해공정 : 나프타 => 탄화수소 가스
- 수소화 분해 공정 : 나프타 => 탄화수소 가스
- 부분연소공정 : 탄화수소 => CO
- 접촉분해 공정 : CO + H => CH4
- 대체천연가스 공정 : CO + H => CH4
#### 열분해 공정 가열방식에 따른 분류
- 외열식 : 원료가 들어 있는 용기를 외부에서 가열하는 방식
- 축열식 : 축열실을 충분히 가열한 후 원료를 통과 시키는 방식
- 부분연소식 : 원료의 일부를 태워 그 열로 나머지 원료를 분해하는 방식
- 자열식 : 외열식+부분연소식
#### 도시가스 제조시 원료 송입방법
- 연속식 : 원료가 연속해서 공급되는 방식 가스로 연속해서 생산
- 배치식 : 일정량의 원료를 반응실에 공급하고 일정량의 가스를 생산
- 사이클릭 (Cyclic)식 : 연속식과 배치식의 혼합 방식
#### 도시가스 제조 프로세스에서 가스화 촉매에 요구되는 성질
- 활성이 높을 것
- 수명이 길것
- 가격이 저렴할 것
- 열, 황, 카본에 내성이 있을 것
#### 도시가스 가스화 프로세스에서 촉매의 피독현상
- 황 성분에 의하여 촉매의 활성도가 떨어지는 현상
#### 도시가스 제조중 가스의 열량 조정방식
- 유량비율 제어방식 : 단순히 유량 비율(원료량:산출량비) 을 조절
- 캐스케이드 방식 : 열량계 신호를 통해 열량을 조절
- 서멀라이저 방식 : 온도제어를 통해 제조가스의 열량을 조절
#### 도시가스 열량조정 공정중 증열법과 희석법을 설명
- 증열법 : 발열량이 낮은 제조가스에 LPG같은 고발열량 가스를 추가
- 희석법 : 발열량이 높은 제조가스에 공기를 희석
## 도시가스 공급설비
#### 도시가스 공급 방식
| **구분** | **압력 범위**                   | **주요 경로**                   |
| ------ | --------------------------- | --------------------------- |
| **고압** | 1.0 MPa 이상                  | 한국가스공사 주배관 (인수기지 ~ 공급관리소)   |
| **중압** | 0.1 MPa 이상 ~ 1.0 MPa 미만     | 공급관리소 ~ 지구 정압기 ~ 도시가스 주배관   |
| **저압** | 0.1 MPa 미만 (통상 2.3~3.3 kPa) | 지역 정압기 ~ 가정용 내관 ~ 가스레인지/보일러 |
#### 1일 수요예측에 의해 여러 종류의 제조설비를 가동할때 가스량 확보 측면에서 고려할 사항
- 경제성 : 저렴한 시설이 일정수준 공급하도록 유지
- 간선 수송능력 : 공급이 중단되지 않도록 경제성 상관없이 주변 설비를 가동
- 가스의 연속성 : 여러 설비의 제조가스가 호환되는지 고려
- 원료 균형 : 여러 원료를 균형 있게 활용
#### 가스도매사업의 가스공급시설의 시설기준
- LPG 저장설비와 처리설비는 그 외면으로부터 보호시설까지 30m
- 가스공급시설은 그 외면으로부터 화기를 취급하는 장소까지 8m 이상의 우회거리
- 고압의 가스공급시설은 안전구획 안에 설치하고 그 안전구역의 면적은 2만㎡ 미만일 것
- 그 외면으로부터 다른 안전구역 안에 있는 고압인 가스공급시설의 외면까지 30m 이상
- 제조소가 인접하여 있는 경우의 가스공급시설은 그 외면으로부터 다른 제조소의 경계까지 20m 이상
- LNG 저장탱크는 그 외면으로부터 처리능력이 20만㎥ 이상인 압축기까지 30m 이상
#### 도시가스시설 기밀/내압 시험 압력
- 최고사용압력의 1.1배와 8.4Kpa중에 높은 압력이상
- 중압이상배관 내압시험 압력 : 최고사용압력의 1.5배

#### LNG 기화장치 종류
- 오픈랙(Open Rack) 법 : 패널 외벽에 해수를 흘려보내고, 패널 내부로 LNG를 통과시켜 열교환
           - 초기 투자 크지만 운영비 저렴, 대용량 기화
- 서브머지드(Submerged) 법 : 가열한 온수통에 LNG 튜브를 통과시켜 열교환
             - 겨울철에 오픈랙 대신 사용, 운영비 비쌈  
- 중간매체법 : 열원이 중간매체를 끓이고, 그 증기가 상부의 LNG 튜브를 감싸며 열을 전달
## 가스홀더
#### 가스홀더의 기능(역할)
- 수요의 시간적 변화에 대해 공급량을 확보
- 공급시설 중단에 대해 공급량 확보
- 소비지역 근처에 두어 피크시 공급 효과
- 공급가스의 조성을 균일화

#### 가스홀더의 종류
- 유수식:** "물그릇에 뒤집어 놓은 컵이 오르내리는 형태"
- 무수식:** "고정된 실린더 안에서 피스톤이 움직이는 주사기 형태"
- 구형 가스홀더(중, 고압)
#### 가스홀더 유해성분 측정
- 매주 1회 (황화수소, 암모니아, 일산화탄소)
