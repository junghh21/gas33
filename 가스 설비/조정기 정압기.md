## LP 가스 조정기
#### LP 가스 조정기 주요기능
- 감압
- 정압
- 폐쇄
#### LP 가스 조정기 종류
- 1단 감압식 저압 조정기
- 1단 감압식 준저압 조정기
- 2단 감압식 1차용 조정기
- 2단 감압식 2차용 조정기
- 자동교체식 일체형
#### LPG 사용시설에서 2단 감압방식 사용시 장점
 - 입상배관 압력 손실을 보정 가능
 - 배관이 길어도 됨
 - 각 연소기구에 맞는 압력 공급
 - 중간 배관 지름 작아도 됨
#### LPG 사용시설에서 2단 감압방식 사용시 장점
 - 설비가 복잡
 - 검사 포인트가 많아 검사 번잡
 - 부탄의 경우 재액화 우려
 - 고압용 이음

#### LPG 조정기 압력
| **구분**         | **입구 압력 범위**    | **조정(출구) 압력** | **폐쇄 압력 (Shut-off)** | **안전장치 작동(개시)**  | **안전장치 정지** |
| -------------- | --------------- | ------------- | -------------------- | ---------------- | ----------- |
| **1단 감압식 저압**  | 0.07 ~ 1.56 MPa | 2.3 ~ 3.3     | 3.5 이하               | 7.0 ± 1.4        | 5.0 이상      |
| **1단 감압식 준저압** | 0.1 ~ 1.56 MPa  | 5 ~ 30        | 조정압력의 1.2배 이하        | 조정압력의 1.7 ~ 2.5배 | -           |
| **2단 감압식 1차용** | 0.1 ~ 1.56 MPa  | 35 ~ 170      | 조정압력의 1.2배 이하        | 조정압력의 1.7 ~ 2.5배 | -           |
| **2단 감압식 2차용** | 0.025 ~ 0.2 MPa | 2.3 ~ 3.3     | 3.5 이하               | 7.0 ± 1.4        | 5.0 이상      |
| **자동교체식 일체형**  | 0.1 ~ 1.56 MPa  | 2.3 ~ 3.3     | 3.5 이하               | 7.0 ± 1.4        | 5.0 이상      |

## 정압기
#### 정압기 기능
- 감압기능 : 1차측 압력을 감압하여 2차측에 공급
- 정압기능 : 2차측 압력을 균일하게 유지
- 폐쇄기능 : 가스 소비가 중단되면 폐쇄하는 기능
#### 정압기 특성
- 정특성 : 유량과 2차압력과의 관계 (로크업, 오프셋, 시프트)
- 동특성 : 부하변동에 따른 제어응답성와 안정성
- 유량특성 : 밸브의 개페와 유량의 관계 (직선형, 2차형, 평방근형)
- 사용최대차압 : 정상 작동 가능한 1차, 2차간의 최대 차압
- 작동최소차압 : 정상 작동 가능한 1차, 2차간의 최소 차압
#### 정압기 정특성 
- 로크업 : 유량이 0일때와 설정압력과 실제압력 차이
- 오프셋 : 유량 변화 시 설정압력과 실제압력 차이 (출구 마찰)
- 시프트 : 1차 압력 변화가 유량, 2차 압력 관계를 전체적으로 이동
#### 정압기의 동특성
- 제어 응답속도가 빠르고 안정성이 나쁜 경우
- 제어 응답속도가 느리고 안정성이 좋은 경우
#### 정압기의 유량특성
- 직선형 : 메인밸브의 개구부가 사각형
- 2차형 : 메인밸브의 개구부가 V자형
- 평방근형 : 메인밸브의 개구부가 접시형
#### 정압기 정특성 히스테리시스 효과
- 유량 증가시 출구 마찰로 압력이 감소
- 유량 감소시 밸브와 다이어프램의 **마찰·탄성 지연**
- 유량 증가시 곡선과 유량 감소시 곡선이 다름
#### 피셔식 정압기(Fisher Regulator) 특징
- 정밀한 압력 제어
- 높은 안정성
- 로딩형 구조
- 유지관리가 용의
#### 피셔식 정압기 작동
- 2차 압력 낮음 => 파일럿 밸브 열림 => 파일럿 구동압력 상승 => 메인 구동압력 증가 => 메인밸브 열림  => 유량 증가
- 2차 압력 높음 => 파일럿 밸브 닫힘 => 파일럿 구동압력 감소 => 메인구동 압력 감소 => 메인밸브 닫힘  => 유량 감소
#### 레이놀즈식 정압기의 특징
- 정특성 우수
- 안정성 부족
- 크기가 큼
- 언로딩 구조
#### 레이놀즈식 정압기 작동
- 2차 압력 낮음 => 파일럿 구동압력 상승 => 메인 구동압력 감소 => 메인밸브 열림  => 유량 증가
- 2차 압력 높음 => 파일럿 구동압력 감소 => 메인 구동압력 증가 => 메인밸브 닫힘  => 유량 감소
#### AFV식 정압기 특징
- 정특성 우수
- 높은 안정성
- 높은 응답성 (파일럿 자동제어)
- 유지관리가 어려움
#### AFV식 정압기 작동
- 2차 압력 낮음 => 파일럿 밸브 열림 => 파일럿 구동압력 상승 => 메인 구동압력 증가 => 메인밸브 열림  => 유량 증가
- 2차 압력 높음 => 파일럿 밸브 닫힘 => 파일럿 구동압력 감소 => 메인구동 압력 감소 => 메인밸브 닫힘  => 유량 감소
#### 피셔식 정압기 2차압력 이상 상승원인
- 메인밸브 폐쇄 불량 
- 2차압 조절관 막힘
- 바이패스 밸브 누설
- **파일럿 공급 밸브 누설**
- **메인밸브, 다이어프램, 스프링, 센터스템 이상**
#### 레이놀즈식 정압기 2차압력 이상 상승원인
- 메인밸브 폐쇄 불량 
- 2차압 조절관 막힘
- 바이패스 밸브 누설
- **보조정압기 누설**
#### AFV식 정압기 2차압력 이상 상승원인
- 고무슬리브 폐쇄 불량
- 2차압 조절관 막힘
- 바이패스 밸브 누설
- **파일럿 공급 밸브 누설**
#### 피셔식 정압기 2차압력 이상 저하원인
- 정압기 능력 부족
- 필터의 먼지류 막힘
- 파일럿 오리피스 막힘
- **메인밸브, 다이어프램, 스프링, 센터스템 이상**
#### 레이놀즈식 정압기 2차압력 이상 저하원인
- 정압기 능력 부족
- 필터의 먼지류 막힘
- **메인밸브, 다이어프램, 스프링, 센터스템 이상**
- **보조정압기 열림 이상**
#### AFV식 정압기 2차압력 이상 저하원인
- 정압기 능력 부족
- 필터의 먼지류 막힘
- 파일럿 오리피스 막힘
#### 정압기의 이상감압에 대처방안
- 저압 배관의 루프화
- 2차측 압력감시장치 설치
- 정압기 병렬 설치
#### 도시가스 사용시설 월 사용량 계산
- 산업용 1일 8시간 사용, 가정용 1일 3시간 사용
- 1월은 30일
- 시간당 열량 * 8 or 3 * 30 = 월 총 열량
- 총 열량 / 단위 부피당 열량 => 월 총 용량 $m^3$
