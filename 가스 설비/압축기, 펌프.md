## 용적형 압축기
#### 용적형 왕복동식 압축기 특징
- 용적형, 피스톤 왕복, 맥동현상 (소음, 진동)
- 고압이 쉽게 형성, 압축효율이 높다
- 오일 혼입 우려
- 구조가 복잡, 수리가 어렵다.
- 흡입밸브, 토출밸브 필요, 설치면적이 크다
#### 용적형 회전식 압축기 특징
- 용적형, 날개(Vane) 회전
- 연속적인 토출, 맥동현상이 없다
- 소형, 주로 소용량에 사용
- 구조가 간단하여 구동이 쉬움
- 직결시 고압축비
#### 용적형 스크류(나사) 압축기
- 용적형, 암수로터의 맞물림
- 연속적인 토출, 맥동현상이 없다
- 효율이 낮고 용량 조절이 어렵다
- 소음발생
#### 왕복동형 압축기의 흡입, 토출 밸브 구비조건
- 개폐가 확실, 작동이 양호
- 유체 저항이 적을 것
- 누설이 없고 
- 파손에 강하고 운전중 분해되지 않을 것
#### 왕복압축기의 체적효율에 영향을 주는 요소
- 톱 클리어런스
- 사이드 클리어런스
- 냉각 불량
- 기체 마찰
- 누설
#### 왕복압축기에서 톱클리어런스가 크면 영향
- 체적효율이 떨어짐
- 압축온도 상승
- 토출가스 온도 상승
- 동력소모 많아짐
- 윤활유 열화
## 터보형 압축기
#### 터보형 원심식 압축기
- 원심형, 임펠러=>디퓨져
- 연속적인 토출, 맥동현상이 없다
- **서징(Surge)** 현상
- 효율이 낮고 용량 조절이 어렵다
- 고속회전, 모터직결
- 설치면적이 작다.
#### 터보형 원심식 압축기의 서징현상
- 유량감소, 토출 압력 증가 => 역류 발생 => 반복
#### 터보형 원심식 압축기의 서징현상 방지법
- 흡입량 조절 (교축밸브:Throttling Valve)
- 방출측 역류방지 밸브
- 베인 컨트롤에 의한 방법
- 회전수 조절
#### 터보형 축류 압축기
- 대형
- 다단 압축용이
- 구조가 복잡
## 일반
#### 압축기 운전 개시전 점검사항
- 압력계, 온도계 확인
- 냉각수, 윤활유 점검
- 볼트의 조임 상태 확인
- 밸브 확인
#### 압축기 운전중 점검사항
- 압력 이상 유무
- 온도 이상 유무
- 누설 유무
- 진동 유무
- 이상음 유무
#### 압축기 가동중 실린더에서 이상음 발생 원인
- 실린더 피스톤 접촉
- 피스톤링 파손
- 실린더 라이닝 파손
- 불순물 혼입
- 액압축
#### 압축기의 과열 원인
- 가스량 부족
- 압축비 증가
- 냉각수 부족
- 윤활유 부족
#### 압축기의 압축비가 증대될 때 나타나는 현상
- 체적효율 감소
- 소요동력 증가
- 압축기 능력 감소
- 토출량 감소
- 토출온도 상승
- 윤활유, 부품 수명 감소
#### 압축기 토출온도 상승시 원인
- 실린더 흡기밸브 이상
- 실린더 배기밸브 이상
- 압축비 상승
- 냉각기 고장- 
#### 왕복동형 압축기의 실린더를 냉각할 때 나타나는 현상
- 체적효율 증가
- 소요동력 감소
- 압축기 능력 증가
- 토출량 증가
- 토출 온도 감소
- 윤활유, 부품 수명 증가
#### 압축기 다단 압축의 목적
- 유체 과열 방지
- 체적 효율 증가
- 압축 일이 감소
- 힘 평형을 좋게
#### 압축기 단수를 결정하는데 고려할 사항
- 토출압력
- 취급가스량
- 취급가스종류
- 연속운전 여부
- 동력 및 제작 경제성
#### 압축기/펌프의 상사의 법칙
- 풍량(유량) : 임펠러 회전수에 비례 ($Q_2 = Q_1 \times (\frac{N_2}{N_1})$)
- 풍압(압력,양정) : 임펠러 회전수 제곱에 비례 ($H_2 = H_1 \times (\frac{N_2}{N_1})^2$)
- 동력 : 임펠러 회전수 세제곱에 비례 ($P_2 = P_1 \times (\frac{N_2}{N_1})^3$)
#### 압축기 윤활유의 구비조건
- 내열성 (인화점 높고) 
- 부동성 (응고점 낮고)
- 화학적으로 안정
- 적당한 점도 유지
- 적은 불순물 함유
#### 압축기별 윤활유 재료
- 산소압축기 : 물, 묽은 글리세린수
- 염소 압축기 : 진한 황산
- LP가스 압축기 : 식물성유
- 공기 압축기 : 양질의 광유
- 수소 압축기 : 양질의 광유
- 아세틸렌 압축기 : 양질의 광유
- 아황산가스($SO_2$) : 화이트유
- 염화메탄 : 화이트유
#### 산소압축기 윤활류로 쓸 수 없는 것
- 광유
- 식물성유, 동물성유
- 합성유 (탄화수소)
## 압축기 용량 제어법
#### 압축기에서 용량제어를 하는 목적
- 수요, 공급의 균형
- 압축기 보호 (경부하 운전)
- 동력 낭비 방지
#### 왕복동식 압축기 연속적인(무단계) 용량 제어법
- 타입드밸브 흡입 제어
- 회전수 변경
- 바이패스 밸브 (흡입측으로 바이패스)
#### 왕복동식 압축기 단계적(실린더 선택)인 용량 제어법
- 클리어런스 밸브
- 실린더별 흡입밸브 제어
#### 터보(원심식)형 압축기의 용량 제어법
- 속도제어에 의한 방법
- 토출밸브에 의한 방법
- 흡입밸브에 의한 방법
- 바이패스에 의한 방법
## 용적식 펌프
#### 용적식 왕복펌프의 특징
- 토출량이 일정
- 소형
- 고압발생 용이
- 고점도 유체 적합
- 맥동현상 발생
- 고압으로 액성질 변화 우려
#### 용적식 왕복펌프의 종류
- 피스톤 펌프 : 용량이 크고 낮은 양정
- 플런저(Plunger) 펌프 : 용량이 적고 높은 양정
- 다이어프램 펌프 : 불순물 많은 유체
#### 용적식 회전펌프
- 날개회전에 의한 원심력 이용
- 연속적인 토출, 맥동현상이 없다
- 점성 유체에 적합
#### 용적식 회전펌프의 종류
- 기어식
- 나사식
- 베인식
## 터보식 펌프
#### 원심 펌프의 특징
- 원심력으로 유체를 압송
- 고양정에 적합
- 소형으로 설치면적이 적다
- 서징, 캐비테이션 현상 발생
- 흡입, 토출밸브가 없다.
- 맥동현상이 없다.
#### 원심펌프의 종류
- 볼류트 펌프
- 터빈 펌프
#### 원심펌프의 축봉장치
- 그랜드 패킹 : 누설되어도 무방한 경우
- 메커니컬 실 (밸런스 실) : 가연성, 독성 액체
#### 원심펌프의 운전특성
- 직렬운전 : 양정 증가, 유량 일정
- 병렬운전 : 양정 일정, 유량 증가
#### 사류 펌프
- 액체의 흐름이 축에 대해 비스듬히 토출되는 형식
#### 축류 펌프
- 액체의 흐름이 축과 일치하게 토출되는 형식
## 펌프에서 발생되는 현상 (유량, 양정, 회전수, 온도)
#### 펌프에서 발생되는 현상 종류와 설명
- 캐비테이션 현상 : 유속증가 => 압력강하 => 기포 발생 => 터지면서 충격파
- 수격작용 : 유속변화 => 압력 급변 => 소음과 진동 발생
- 서징(맥동)현상 : 토출 압력과 유량에 주기적인 변화가 발생
- 베이퍼록 현상 : 온도상승 => 기포가 관로를 막아 흐름 차단
#### 펌프에서 발생되는 캐비테이션 현상 원인
- 양정 : 흡입양정이 큰경우
- 유량 : 흡입 저항이 증가
- 회전수 : 임펠러 회전수가 높은 경우
- 온도 : 유체 온도 상승
#### 펌프에서 발생되는 캐비테이션시 일어나는 현상 
- 소음, 진동
- 임펠러 침식
- 효율 저하
- 양수 불능
#### 펌프에서 발생되는 캐비테이션 현상 방지법
- 양정 : 양정을 짧게
- 유량 : 양흡입펌프, 병렬운전
- 회전수 : 회전수를 낮춘다.
- 온도 : 흡입 유체의 냉각
- 임펠러가 완전히 잠기게 한다.
#### 펌프에서 발생되는 수격(Water Hammering) 현상 원인
- **급격한** 밸브의 개폐
- **급격한** 펌프의 정지
- **급격한** 유속의 변화
#### 펌프에서 발생되는 수격(Water Hammering) 현상 방지법
- 밸브를 송출구 가까이 두고 적절히 제어
- 배관 지름을 크게
- 서지탱크 설치
#### 펌프에서 발생되는 서징(맥동) 현상 원인
- 양정-유량 특성 변곡점 운전
- 배관 내에 공기 주머니
- 유량조절 밸브가 탱크 뒤에 있을때
#### 펌프에서 발생되는 서징(맥동) 현상 방지법
- 임펠러 변경
- 공기 제거
- 밸브 위치 조정
#### 펌프에서 발생되는 베이퍼록 현상 원인
- 온도 : 온도 상승 (외부열원)
- 압력 : 배관이 좁을때, 고양정
#### 펌프에서 발생되는 베이퍼록 현상 방지법
- 온도 : 배관 냉각
- 압력 : 흡입 배관을 크게, 양정을 작게
#### 저비점 액체용 펌프 사용시 주의사항
- 펌프는 탱크 가까이
- 흡입, 토출측에 신축조인트 설치
- 펌프와 밸브 사이에 안전밸브 설치
- 펌프를 충분히 냉각
#### 펌프에서 토출량이 감소하는 원인
- 임펠러 마모
- 임펠러 이물질
- 공기 흡입
- 케비테이션 발생
- 관로저항 증가
#### 펌프 전동기 과부하 원인
- 양정 증가
- 유량 증가
- 액 점도 증가
- 액비중 증가
- 임펠러에 이물질 유입
#### 펌프에서 이상음 발생 원인
- 임펠러 이물질
- 공기 흡입
- 케비테이션 발생
- 서징 발생
