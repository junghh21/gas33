## 금속 일반
#### 금속재료 정의
- 크리프(Creep)현상 : 하중이 가해진 채 오래되면 변형이 증가
- 가공경화 : 가공할수록 경도가 증가
- 청열취성 : 탄소강에 고온에서 경화되어 취성을 갖는 현상
- 피로파괴 : 반복적인 낮은 응력에 파괴되는 현상
#### 탄소강 성분 (Fe, C외)
- 인 (0.04%)
- 황 (0.05%)
- 규소
- 망간
#### 열처리의 종류
- 담금질 : 급랭, 강도 경도 증가
- 풀림(Annealing) : 서냉(로내), 재질 연화, 내부응력 제거
- 불림(Normalizing) : 서냉(공기), 결정조작의 미세화, 표준화
- 뜨임(Tempering) : 변태점이하 서냉, 연성, 인장강도
#### 용접이음부 잔류응력 제거방법
- 열처리 풀림
- 기계적 응력 완화방법
- 저온 완화 방법
- 쇼트 피닝
## 금속 취성
#### 금속의 저온특성
- 경도, 인장강도 증가
- 연성, 수축률 감소
#### 탄소강의 저온 취성
- 탄소량이 많을수록
- 불순물(인, 황)이 많을수록
#### 금속 중 저온 취성이 거의 없는 것
- 스테인리스강
- 구리
- 알루미늄

## 금속 부식
#### 금속부식 정의
- 갈바닉 부식 : 전위가 낮은 금속이 양극이 되는 현상
- Corrosion (부식, 腐蝕) : 금속이 전해질 속에서 양극화되어 산화
- Erosion (침식, 蝕) : 물리적 충격(흐름) 표면을 깎음 (굴곡 배관, 임펠러)
- 입계부식 : 고온에서 스테인레스강의 크롬이 탄화되며 부식이 쉬운 상태로 변함
- 바나듐 어택: **바나듐** 연소 후 액체화되어 금속 부식 (**고온 부식**)
#### 금속부식의 원인 (재료, 응력, 온도, 전위차)
- 이종금속 접촉
- 조성 불균일
- 표면상태 불균일
- 잔류응력
- 고온, 저온, 수분
#### 고압장치 금속재료의 부식 방지 대책 
- 부식 환경 처리 (고온/저온, 수분, 응력)
- 부식억제제
- 전기 방식
- 피복
#### 금속재료의 이종금속 접촉 부식 방지
- 전위차가 작은금속끼리
- 전기방식법
#### 금속재료의 응력부식 방지 대책
- 잔류응력 제거
- 합금 조성 조정
- 부식 환경(수분) 처리
#### 가스(고온)에 의한 금속 부식
- 산화 : 산소, 이산화탄소,   조건: 고온고압, 방지: Cr, Al, Si
- 황화 : 황화수소 ($H_2S$),     조건: 수분,       방지: Cr, Al, Si
- 질화 : 암모니아 ($NH_3$),    조건: 고온고압, 방지: Ti, V, Al, Si
- 카르보닐화 : 일산화탄소, 조건: 고온고압, 방지: Ni
- 탈탄 : 수소,                   조건: 고온고압, 방지: Ti, V, Mo, Cr
#### 고압장치 금속재료중 고온/저온재료 구비조건
- 내열, 내냉
- 내부식성
- 강도가 클것, 내마모성
- 가공이 용이하고 가격 저렴
#### 금속재료의 충격 시험 장비와 목적
- 충격시험기
- 금속의 인성과 취성 시험
